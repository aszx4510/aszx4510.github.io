這周一(11/27)去 IJCNLP 的 workshop (DDDSM) 報 paper，
簡單記錄一下心路歷程XD
順便記錄 IJCNLP Day 0 的感想。

Oral Presentation

題目

〈 Incorporating Dependency Trees Improve Identification of Pregnant Women on Social Media Platforms 〉
就是在 About Me 中提到過的研究，
利用推特內容判斷 user 是否表達已經懷孕的資訊，
像是下面這則 tweet，充分表示出她已懷孕，而且還附上懷孕照片，
毫無疑問，這是個 positive case。
懷孕婦女的推特

事前準備

  • PPT
  • 講稿
  • 練習、試報

投影片和講稿都是必要的東西，

投以片一開始字體太小、字數太多，
講稿的部分，一開始我還不想準備講稿，
想試著即興發揮，但我英文太爛了，
如果沒有準備講稿肯定爆炸。

最後練習試報，總共練習了三次，
第一次是Morphe，第二次是Morphe和乃文，
第三次是書豪，這次還是用Line報給他聽XD

每次報告完，大家都說「還不錯」、「很好/很棒」，「至少以第一次用英文報告來說，很有水準了」，
感謝大家給我許多信心XD 感謝信心加持

實際上台

在503會議室舉行 DDDSM Workshop

11/27，實際上台報告，
在台上報告時真的非常緊張XD
明明台下人數不多，甚至好幾個是自己人，
但我還是沒辦法控制好緊張的情緒。

開始報告!

一手拿麥克風，另一手拿光筆，
實在是沒辦法拿著講稿，
甚至連翻頁也很困難，這部分需要改進。

好險最終按照練習時的報告方法，
順利的報告完畢，有些句子需要看著講稿念，
最後大家討論的滿多的，提出許多問題，
不過我聽不太懂XD (感謝JJ和詠淳幫我回應)。

其實台下人數不多XD


改進

最大問題就是我的英文太爛了😔
如果英文夠好，就不會有這麼多其他問題了，
不過英文能力不是短期就能改善的，只好從其他部分補強了。

主要缺點就是練習次數太少了，應該要練習更多次，
對於講稿內容和投影片順序瞭若指掌，
才能達到不需要講稿，「看起來」像是即興發揮的境界😆

IJCNLP Day 0 心得

IJCNLP 2017 in Taipei

第一次參加 international conference，而且還要上台報告，
雖然很緊張，但總結來說還是挺新鮮的，哈哈，
而且我發現，外國人的英文也不見得很流利:p

我們實驗室有許多人參加 DDDSM Workshop，
除了我，還有 Onkar、白熊、Neha 也報告了各自的研究，
白熊的英文口說出乎意料的…不流暢，
但白熊還是相當有自信的報告完全部內容，
這種自信就是我該學習的部分。

食物方面，
coffee break 的點心很好吃，但可惜沒有奶茶>_<
由於是 pre-conference workshop，參加的人數不多，
點心根本吃不完阿。

Day 0 心得就先這樣子吧，
IJCNLP Main Conference 的心得就留到下篇啦~
