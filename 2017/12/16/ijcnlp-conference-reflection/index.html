IJCNLP 2017 in Taipei

上一篇說好的 IJCNLP 心得終於來了:D

IJCNLP Day 1 心得

期待已久的 IJCNLP Main Conference，
第一天就睡過頭啦，好險南港展覽館很近…😔

南港展覽館入口的 IJCNLP 廣告

首先介紹一下會議的的時程規劃，每天由許多 Parallel Session 和 Coffee Break 組成，
每個 Parallel Session 有 4 個 20 分鐘的 long paper presentation，以及 1 個 10 分鐘的 short paper presentation，
所以一個 Parallel Session 總共 90 分鐘，
重點是同時段會有三個 Session 同時進行，要挑選自己想聽的主題。

Coffee Break 就是一堆點心吃到飽，還有咖啡喝到爽XD

Conference Program - Day 1

午餐時間

到會場後，立即與詠淳和Morphe會合，
可惜他們有事情，我只能獨自吃便當了QQ

便當菜色很普通，和前一天的 DDDSM 一樣，就是普通的會議便當，
吃完便當跑去廠商區閒逛，沒想到遇到熟人😆
意藍資訊 eLand — 玖哲學長

據玖哲學長情報，
意藍資訊有贊助 ROCLING，所以能夠在 IJCNLP 廠商區擺放攤位，宣傳公司的產品和研究，
學長在意藍資訊當研替，目前已經是第三年了，
學長說…上班壓力很大，但好處是(幾乎)不用加班。

難得以和業界第一線人員接觸，一定要好好了解國內的 NLP 公司需求到底是什麼，
意藍資訊的 NLP 小組，負責幫客戶分析許多 NLP 的相關資訊，例如產品評論和社群意見，
目前使用方法以傳統 SVM 為主，人工抽取 n-gram 等 feature，也會利用社群對話內容 train word embedding，

除了遇到玖哲學長，也有遇到俊鍇學長和其他 IISR 的新成員，
另外在六樓 ROCLING 的休息區，遇到蔡宗翰老師，
我和老師打招呼、向老師報告我的近況，也聊到將來畢業後的打算，
不過真的是很久沒見到蔡老師了，想不到老師還認得我XD

Parallel Session 2: Machine Translation 2

因為中午才到會場，今天只能聽到下午的 Session 囉 ，這場我選擇 Machine Translation。

現在 Deep Learning 這麼熱門，
Machine Translation 根本變成 Neural Machine Translation (NMT) 了，
以下節錄一些演講筆記。

  1. An Exploration of Neural Sequence-to-Sequence Architectures for Automatic Post-Editing
    使用 encoder-decoder model with attention 的架構，encoder 和 decoder 是 GRU，乍聽之下沒什麼特別的，Github連結。

  2. Imagination Improves Multimodal Translation
    用圖片來輔助機器翻譯，這個 idea 很有趣，我以前有想過這個點子，但從沒想過可行XD
    也是 encoder-decoder model with attention 的架構，Github連結。

  3. Understanding and Improving Morphological Learning in the Neural Machine Translation Decoder
    又是 encoder-decoder model with attention 的架構，還有一些其他技巧，但我聽不太懂，Github連結。

  4. Improving Neural Machine Translation through Phrase-based Forced Decoding
    報告人應該是一位在日本的中國女學生，她念英文速度超級無敵快，完全聽不懂到底在幹嘛XD
    這段完全 pass 😅

  5. Neural Lattice Search for Domain Adaptation in Machine Translation
    利用 phrase-based machine translation 產生 graph (lattice)，然後搜索 graph 上的 pattern ，Github連結。

Poster and Demo Session

第一天下午比較特別，有一個 Poster and Demo Session，
在這個時段，大家會在六樓的大空間中貼海報、介紹自己的研究。
我們實驗室有投稿上兩篇 paper，分別是:

  1. MONPA: Multi-objective Named-entity and Part-of-speech Annotator for Chinese using Recurrent Neural Network
    使用 attention RNN model 替句子一次完成斷詞、標 POS-tag 和 NER。
    Demo 網址

  2. Identifying Protein-protein Interactions in Biomedical Literature using Recurrent Neural Networks with Long Short-Term Memory
    這個我就比較不熟了，主要由 Morphe 學長負責的研究，
    應該是使用 RNN 研究 Protein-Protein Interaction (PPI) 。

雖然兩項都不是我主要負責的研究，不過我在 MONPA 方面提供較多的幫忙，
詠淳學長請我到場支援，一起解說 MONPA 的 model 和特色。

MONPA 使用多層雙向 LSTM encoder，並且搭配 attention 機制，
由 decoder 一個一個 output 每個字的邊界、詞性和 Name Entity，
越來越多研究使用這種架構，不過我們首次將這個架構應用在繁體中文斷詞，取得很好的成績。

特個階段彷彿是認親大會，遇到更多以前的學長和朋友，
首先，一開始幫忙貼海報時就遇到 Journey 學長，
Journey 目前在台達研究院工作，公司補助他和其他同事參加 IJCNLP，
感覺是很不錯的公司，明年畢業後應該會去面試看看。

後來還有遇到韓文彬，清大張俊盛老師的 lab 也有參加 IJCNLP，
好久沒見到他了，沒想到跑去張老師的實驗室XD

整個 session 過程非常歡樂，有時遇到熟人閒聊一下聯絡感情，
也有許多人針對 MONPA 深入提問和給予建議，
相當多人關心 NLP 的發展呢！
