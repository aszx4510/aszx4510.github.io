研究專題

Topic Classification by Template Convolution Neural Network

目前正在做的專案，從7月開始的研究，預計作為碩論主題，
Task 目標是新聞主題(topic)分類，總共六類，
特色是使用 CNN 做文件分類，而且是 sense-based model，
沒錯，不是 character-based，也不是 word-based，
希望使用 CNN 的 filter 說明/證明 pattern (sense sequence) 的重要性，
目前成果還不錯，不過希望可以再提升一些 performance更好。

Identification of Pregnant Women on Social Media

2017年7~8月時做的小task，目標是辨識 Tweets 的 user 是否傳達出她已懷孕的資訊，
這個研究不只和 NLP 相關，也和 health-care 有點關聯，
使用 KeLP 的 SPTK 架設 model，F1-score 比傳統 Tree Kernel 高 6%，
而且寫了一篇 paper 投到 IJCNLP 的 workshop (DDDSM)，
2017/11/27 oral presentation。

Machine Translation and OpenMT15

2014下半年做的專題，使用 Moses 建立 model，
結合 translation model 和 language model 來翻譯，
後來有用此 model 參加 OpenMT15 這個比賽，
排名一般而已XD
